# Hangman 

This is a simple game made is javascript using Jade, Stylus and AngularJS.

### Hot to run locally?

1. Clone repository
1. Install
```sh
$ npm install
```
1. Run server
```sh
$ gulp server
```
1. Open URL: http://localhost:8000
```sh
$ open http://localhost:8000
```
